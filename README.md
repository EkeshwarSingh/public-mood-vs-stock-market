# Public Mood vs Stock Market

## Overview
This project investigates whether public sentiment from financial news can explain or predict NIFTY 50 stock market movements.

## Tasks Covered
- Web Scraping
- Exploratory Data Analysis (EDA)
- Data Visualization
- Sentiment Analysis

## Tech Stack
- Python
- Pandas
- BeautifulSoup
- VADER Sentiment
- Matplotlib
- Seaborn
- Scikit-learn
- Power BI
